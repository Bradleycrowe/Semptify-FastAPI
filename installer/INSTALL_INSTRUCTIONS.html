⚙ Automatic Installation

The easiest way to install Semptify:

Locate the Installer

Find Install-Semptify.bat in the extracted folder

Run as Administrator

Right-click the file and select "Run as administrator"

Follow the Prompts

The installer will:

  • Install Python 3.12 (if not present)
  • Install PostgreSQL 16 (if not present)
  • Create virtual environment
  • Install all dependencies
  • Configure the database
  • Create desktop shortcut
Advanced Options: Run PowerShell as administrator and use: 
.\install_semptify.ps1 -InstallService    # Auto-start on boot
.\install_semptify.ps1 -InstallPath "D:\Semptify"   # Custom location

🛠 Windows Service (Auto-Start)

To have Semptify start automatically when Windows boots:

# Install as service (requires NSSM)
.\install_semptify.ps1 -InstallService

# Or manually with NSSM:
nssm install Semptify "C:\Semptify\Semptify-FastAPI\.venv\Scripts\python.exe"
nssm set Semptify AppParameters "-m uvicorn app.main:app --host 0.0.0.0 --port 8000"
nssm set Semptify AppDirectory "C:\Semptify\Semptify-FastAPI"
nssm start Semptify

Service commands:

Start-Service Semptify    # Start the service
Stop-Service Semptify     # Stop the service
Restart-Service Semptify  # Restart the service

🔧 Troubleshooting

"Python not found"

Install Python from the included python/ folder, or download from python.org

python\python-3.12.0-amd64.exe

"PostgreSQL connection failed"

Check if PostgreSQL service is running:

Get-Service postgresql*
Start-Service postgresql-x64-16

"Port 8000 already in use"

Find and stop the process using port 8000:

netstat -ano | findstr :8000
taskkill /PID [process_id] /F

Or use a different port:

.venv\Scripts\python.exe -m uvicorn app.main:app --port 8080

"Cannot connect to database"

Verify the database exists:

cd "C:\Program Files\PostgreSQL\16\bin"
psql -U postgres -l

Create it if missing:

psql -U postgres -c "CREATE DATABASE semptify;"

Reset Database Password

cd "C:\Program Files\PostgreSQL\16\bin"
psql -U postgres -c "ALTER USER postgres PASSWORD 'Semptify2024!';"
